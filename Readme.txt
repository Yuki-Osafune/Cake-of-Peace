・このファイルの中身について

　こちらは、3Dシューティングゲーム、Cake-of-Peaceという作品になっております。ゲーム制作プロジェクトという講義にて、3人1組のチームで作成した作品となります。
3人1組で作成したため、仲間たちで役割を分担して作業させていただきました。
　私はチームのリーダーとして指揮を執り、全体のプログラミングの一部とゲーム内のミニマップ、UI、タイトル画面などを担当させていただきました。
　（残りの一人は他のプログラミングとモデルの作成を、もう一人も他のプログラミングを担当しました。）
　なお、私が担当したプログラムは、「カメラの制御」「プレイヤーの動き」「タイトル画面と推移」となります。ただ、今回このゲームのビルドが出来なかったのが心残りともいえます。

　ゲームのルールは、中央のケーキに向かって周囲から敵が襲い掛かってきます。その敵たちを主人公は持っている銃で倒しながら、中央のケーキを守り、何WAVEまでたどり着けるかが目標となります。また、敵はケーキまでたどり着くと、ケーキを食べ始めて残量が減っていきます。そして、残量が無くなってしまうとゲームオーバーになります。（操作説明は、タイトル画面に記入されています。）

　プロジェクトの一環として作成したため、１か月ほどの期間をかけて作成することとなり、その間リーダーとして活動したため、その分苦労のあった作品ともいえます。
　例えば、仲間たちどの作業をやらせるか、仲間たちの意見を聞きながら割り振るように作業した事。作業状況を把握するため情報交換を繰り返すようにしたこと。自身の作業でも、様々な情報を参考にしながら作業し、その際仲間たちにも情報を共有するように心がけた事がありました。

　反省点としては、期間の関係もあったが、いくつか不備を残した状態で終わってしまったのが反省点だと言えます。
　例えば、本来実装する予定の別機能を実装できずに終わってしまったり、ファンシーな世界観にもかかわらず、効果音が世界観に合わないものであったり、ビルドが出来ていないなど、いくつか反省点が残る作品にもなりました。
